`relay help`
============

Display help for relay.

* [`relay help [COMMAND]`](#relay-help-command)

## `relay help [COMMAND]`

Display help for relay.

```
USAGE
  $ relay help [COMMAND] [-n]

ARGUMENTS
  COMMAND  Command to show help for.

FLAGS
  -n, --nested-commands  Include all nested commands in the output.

DESCRIPTION
  Display help for relay.
```
